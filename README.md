# JavaFTPServer

## Description
The **JavaFTPServer** is a project that shows how to implement a FTP server using Apache MINA.

## Technologies
The project uses the following technologies:

* Java 11
* Apache Maven 3.6.3
* Apache MINA 1.1.1
* IDE Eclipse
